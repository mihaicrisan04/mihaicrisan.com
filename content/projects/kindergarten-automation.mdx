A Google Sheets + Apps Script automation built for a local kindergarten that was still running on paper.

## The problem

Teachers were burning the first hour of every morning on the same routine — hand-writing daily behaviour notes, photocopying them, slipping each one into the right child's folder to send home. The admin side was a tangle of spreadsheets that nobody trusted to be up to date, so the same data was re-entered in two or three places.

## What I built

- one Sheets workspace as the single source of truth — attendance, daily notes, behaviour, contacts
- an Apps Script that generates each child's daily report from the sheet and emails it to the assigned parent in one batch send
- monthly admin reports derived from the same data, so there's no double entry

## Why this shape

The kindergarten couldn't justify a SaaS subscription, and the staff already lived inside Google Sheets. Apps Script meant zero hosting cost, zero per-seat licensing, and the spreadsheet they were already maintaining became the system — no new tool to learn, just the existing one with the manual work taken out.
